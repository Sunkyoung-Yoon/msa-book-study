## :star: Pull Request
* PR은 포크 저장소 이용!
```
PR 제목: docs: 1_1 제목 - 이름 형태로 정리글 업로드
ex) docs: 1_1 마이크로서비스 아키텍처로 진화 - 윤선경

맡은 파트 모아서 업로드할 경우
ex) docs: 1_1 ~ 1_6 - 이름

파일명: 요약할 글 목차.md
ex) 1_1.md
```
